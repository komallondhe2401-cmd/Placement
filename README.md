# 🎓 Student Placement Prediction 
## 📌 Project Overview
This project predicts whether a student will be placed or not based on two features:
- CGPA
- IQ
A Machine Learning model is built using Logistic Regression to classify students into **Placed** or **Not Placed** categories.

## 🚀 Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## 📊 Project Workflow

### 1. Data Preprocessing
- Loaded dataset
- Checked for missing values
- Split data into features (X) and target (y)

### 2. Exploratory Data Analysis (EDA)
- Visualized data using scatter plot
- Analyzed relationship between CGPA, IQ, and placement

### 3. Feature Scaling
- Applied scaling to normalize feature values
- Improved model performance

### 4. Model Building
- Used Logistic Regression algorithm
- Trained model on training dataset

### 5. Model Evaluation
- Evaluated model using accuracy score
- Achieved accuracy around **0.8 – 0.9**

### 6. Decision Boundary Visualization
- Plotted decision boundary
- Visualized how model separates classes

## 📷 Output
- Scatter Plot of CGPA vs IQ
- Decision Boundary visualization

## 🎯 Result
The model successfully predicts student placement and provides a clear understanding of classification using decision boundaries.
